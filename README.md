# Course Recovery System

A Java Swingâ€“based Course Recovery System developed for university coursework.  
The system supports the management of students eligible for course recovery using file-based data storage.

Two user roles are implemented: **Course Admin** and **Academic Officer**.

---

## User Roles

**Course Admin**
- Manage course recovery plans and milestones
- View eligible students
- Monitor recovery progress

**Academic Officer**
- Review student academic records
- Track recovery outcomes
- View course recovery reports

---

## Features
- Java Swing GUI
- Role-based access control
- File-based data persistence (TXT files)
- Course recovery tracking and reporting

---

## Technologies
- Java
- Java Swing
- Object-Oriented Programming
- File handling

---

## Notes
- This system uses text files instead of a database
- Designed for academic and learning purposes
